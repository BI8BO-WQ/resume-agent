# 岗位定制简历投递助手 Agent 分享说明

## 你要发给别人什么

发送整个分享压缩包即可，不要单独只发 HTML。

原因：PDF 简历解析、JD 图片识别、模型分析、官方岗位刷新都需要本地后端 `server.py`。只双击 `index.html` 只能看到部分前端界面，很多核心能力会失败。

## 对方怎么运行

1. 解压压缩包。
2. 确认电脑已安装 Python 3.10 或以上版本。
3. 双击 `start_agent.bat`。
4. 第一次启动会自动生成 `.env`，在里面填对方自己的 `OPENAI_API_KEY`。
5. 浏览器会自动打开：

```text
http://127.0.0.1:8787
```

## 一定不要发送的文件

不要发送你自己电脑里的 `.env` 文件，因为里面可能有真实 API Key。

如果对方要完整使用模型能力，需要使用他自己的 API Key，或者由你们公司统一提供一个可共享的测试 Key/网关配置。

## 如果没有 API Key

页面可以打开，部分本地规则能力还能看，但以下功能会受影响：

- 模型分析岗位匹配
- PDF 复杂解析
- JD 图片 OCR
- 自动提取简历信息

## 常见问题

- 如果提示端口被占用：打开 `.env`，把 `PORT=8787` 改成其他端口，例如 `8788`。
- 如果提示 Python 找不到：安装 Python，并勾选 `Add Python to PATH`。
- 如果模型报 401：API Key 不正确，或没有该模型/网关权限。
